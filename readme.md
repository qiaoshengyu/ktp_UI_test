## 使用方法：

1. 在config.ini文件中填写账号、密码、上传文件的路径（不能有中文）
2. 默认无头模式运行，可修改config.ini中配置文件

## 依赖库：

selenium	pyautogui	python3-xlib

## 注意：

```
linux使用pyautogui时：
需要安装库                    pip3 install python3-xlib
控制X server访问权限运行           xhost +
使用DISPLAY命令运行python        DISPLAY=:0 python3 run.py 
```
